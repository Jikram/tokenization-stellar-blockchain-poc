# Wallet & Custody

This demo uses Freighter for wallet interaction on Stellar Testnet.

Freighter provides a user-owned keypair experience for public demos.

For enterprise custody, the same approval-control concept can be extended to custodial services such as Fireblocks, Blockdaemon, Taurus, or Anchorage.

The key design point is that private keys remain off-chain and are never embedded in frontend code.
