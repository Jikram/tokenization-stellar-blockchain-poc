# Demo Script

This document describes the user-facing flow for the public Stellar Soroban demo.

1. Open the frontend app on a browser configured for Testnet.
2. Connect using Freighter.
3. Confirm the connected wallet address and contract ID.
4. Check the current approval status for the connected wallet.
5. If the connected wallet is the admin, approve a second wallet address.
6. Switch to the approved wallet or provide it as the action actor.
7. Execute the protected action and verify the transaction hash and activity panel.
8. Review emitted events in the activity panel.

The demo illustrates an approval control workflow while keeping private keys local to Freighter.
